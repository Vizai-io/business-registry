# Registry Entry Standard — Retired

The geo-based `registry-entry` contract was retired on 2026-07-16 by BR-02.

Use:

- [Entity Profile Standard](entity-profile-standard.md)
- [Entity Profile Schema](../schema/entity-profile-v1.0.schema.json)
- [BR-02 Model Convergence](migrations/br-02-model-convergence.md)

The only supported production path is:

```text
registry/<entity-slug>/profile.json
```

This tombstone is retained so existing documentation links explain the
migration rather than silently failing.
