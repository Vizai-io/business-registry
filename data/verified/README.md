# Verified Business Profiles

This directory contains business profiles that have completed VizAI's verification process.

## Verification Standards

Verified entries meet these criteria:

- ✅ **Domain Ownership Verified** - Business controls the domain they claim
- ✅ **Human Review Completed** - VizAI analyst has reviewed all information
- ✅ **Sources Documented** - All claims trace back to verifiable sources
- ✅ **Accuracy Monitoring** - Regular checks to detect information drift
- ✅ **Quality Score ≥ 85** - Meets high data quality standards

## Directory Structure

Verified profiles are organized by industry/category:

```
verified/
├── technology/        # Software, SaaS, hardware companies
├── professional-services/  # Consulting, legal, accounting
├── financial/         # Banks, fintech, investment firms
├── healthcare/        # Medical, pharmaceutical, health tech
├── manufacturing/     # Industrial, consumer goods
├── retail/           # E-commerce, brick-and-mortar retail
└── other/            # Businesses not fitting above categories
```

## Verification Process

To get your business verified:

1. **Purchase a VizAI service tier** (Tier 0 or higher)
   - View pricing: https://www.vizai.io/packages.html

2. **Complete verification**
   - Verify domain ownership via DNS or meta tag
   - Provide authoritative source documents
   - Complete human review interview

3. **Profile creation**
   - VizAI creates your profile following schema standards
   - You review and approve before publication
   - Profile added to `/data/verified/`

4. **Ongoing monitoring** (Tier 1+)
   - Regular accuracy checks
   - Drift detection alerts
   - Quarterly profile updates

## Quality Scores

All verified entries include a quality score (0-100):

- **90-100:** Comprehensive, well-sourced, regularly updated
- **85-89:** Complete with good documentation
- **75-84:** Solid core information, some gaps
- **Below 75:** Does not meet verified standards
